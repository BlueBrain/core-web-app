## Installation

```bash
npm install
# or
yarn install
```

### Development

To spin-up a local development server, run the following terminal command.

```bash
npm run dev
# or
yarn dev
```

This will start a local server, which can be accessed at the following port: [http://localhost:3000](http://localhost:3000)

### Testing

```bash
npm run test
# or
yarn test
```
